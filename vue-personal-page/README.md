# Vue Personal Page

这是您个人网站的Vue版本，已完成从传统HTML/CSS/JS到Vue组件化架构的迁移。

## 项目结构

- `src/App.vue`: 根组件
- `src/components/`: Vue组件目录
  - `AppNavbar.vue`: 导航栏组件
  - `AppFooter.vue`: 页脚组件
  - `HomePage.vue`: 主页内容组件
  - `ProfilePhoto.vue`: 个人照片组件
- `src/assets/`: 静态资源目录

## 开发命令

```bash
# 启动开发服务器
npm run serve

# 构建生产版本
npm run build
```
