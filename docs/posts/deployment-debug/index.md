# 🛠️ 部署与调试实战

这里用于整理「代码实际部署运行 Debug」的经验贴，强调可复现、可排障、可回滚。

## 📚 推荐结构

每篇文章建议包含：

1. **背景与目标**：项目是什么，要达到什么可用状态。
2. **环境信息**：系统、运行时版本、依赖版本。
3. **部署步骤**：从拉代码到服务可访问。
4. **问题与排障**：报错、定位过程、根因分析。
5. **修复方案**：最终改动及为什么有效。
6. **复盘总结**：可复用 checklist、后续优化项。
